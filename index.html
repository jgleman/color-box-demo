A Modern JavaScript Color Library

color-box is a set of functions that makes manipulating colors using JavaScript or TypeScript easy.

color-box on GitHub Documentation

color-box Features

Adjust and mix colors. Modular, only use what you need. Includes TypeScript types. Supports tree shaking. Immutable, functions always return a new Color instance.

Example


import { Color, darken, hexString } from "@jgleman/color-box";

const myColor = new Color("#336699");
const myDarkerColor = darken(myColor, 20);
console.log(hexString(myDarkerColor));
// #1a334d
